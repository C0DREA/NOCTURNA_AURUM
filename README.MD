# Nocturna Aurum – Handmade Candle Website

## Overview

Nocturna Aurum is a minimalist, multi-page website designed for a handmade candle brand.
The project focuses on clean UI, responsive layout, and a modern aesthetic.

## Pages

* Home (index.html)
* Shop (shop.html)
* Delivery Info (delivery.html)

## Features

* Responsive design (mobile, tablet, desktop)
* Clean and minimal UI
* Reusable global styling (global.css)
* Product grid layout
* Structured and semantic HTML

## Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* Vanilla JavaScript (basic setup)

## Project Structure

* /styles → CSS files
* /resources → images & icons
* /scripts → JavaScript
* HTML files in root

## Purpose

This project was created as a portfolio piece to demonstrate:

* Front-end fundamentals
* Layout design skills
* Code organization and scalability
